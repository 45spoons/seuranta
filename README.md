# Opt-in kävijäseuranta

## Alustaminen

Vaatimukset:

- python >= 3
- verkkoyhteys tietotekniikkatila-Kattilan lähiverkkoon

```shell
git clone git@github.com:45spoons/seuranta.git
cd seuranta/
python3 -m venv .venv
source ./.venv/bin/activate
pip3 install -r requirements.txt
python3 ./main.py
```
